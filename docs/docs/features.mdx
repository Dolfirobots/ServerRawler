# Features

ServerRawler is packed with features designed to make Minecraft server crawling efficient and comprehensive.

## Core Functionality

-   **High-Performance Crawling:** Optimized for speed using Rust, allowing for rapid discovery of a vast number of Minecraft servers.
-   **Protocol Support:**
    -   **Minecraft Ping Protocol:** Gathers basic server information quickly.
    -   **Minecraft Query Protocol:** Collects more detailed data from servers that support it.
    -   **Join Protocol:** *Currently under development for advanced server interaction.*
-   **Fast IPv4 Generation:** Includes a highly optimized IPv4 generator with CIDR support to efficiently scan IP ranges.

## Data Collection

ServerRawler collects a rich set of data for each discovered Minecraft server, including:

-   **Server Description:** The "Message of the Day" (MOTD).
-   **Player Information:**
    -   Maximum players allowed.
    -   Online player count.
    -   List of online player usernames and UUIDs.
-   **Security Status:** "Enforce Secure Chat" status.
-   **Plugins & Mods:**
    -   List of installed plugins.
    -   Detected mod loaders and associated mods.
-   **Server Configuration:**
    -   Online mode status.
    -   Whitelist information (especially useful for cracked servers).
-   **Version & Software:** Detailed server version and software information.

## Utilities

-   **Base64 to Image Converter:** A built-in utility for handling image data often found in server responses.

## Future Enhancements (Roadmap)

The project is in early alpha, with several exciting features planned:

-   **PostgreSQL Support:** Expanding database compatibility beyond MySQL.
-   **Web API:** Providing an interface for programmatic access to collected data.
-   **Discord Bot Integration:** Control and monitor the crawler via a Discord bot.
-   **Proxy Support:** Enhancing anonymity and reach for crawling operations.
